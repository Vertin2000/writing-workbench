# 工作台基座验收

这份清单检查工作台本身，不检查某一章草稿内容。每次大修前后跑一遍，避免目录还在、结构已经失效。

- [ ] `00_Context` 仍然小而稳定，没有塞入只服务单章的大段素材。
- [ ] `02_Modules` 的任务书能直接指导写作，包含定位、必读材料、最小交付物、不得写入内容和验收清单路径。
- [ ] `03_DataRoom` 有索引，素材都能被模块任务书或草稿解释用途，没有无人引用的材料。
- [ ] `04_Drafts` 能追溯到数据口径、模块任务书和文风要求，文件名保留版本号。
- [ ] `05_Checklists` 覆盖模块级验收和全篇一致性验收，新增问题已沉淀为可复用检查项。
