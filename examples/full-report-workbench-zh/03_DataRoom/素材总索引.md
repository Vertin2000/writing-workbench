# 素材总索引

本层只放按需加载的素材入口。当前 full example 使用合成数据，不包含外部原始文件。

| 素材 | 路径 | 用途 |
| --- | --- | --- |
| 阶段指标 CSV | `03_DataRoom/generated/stage_metrics.csv` | 可机器读取的合成指标 |
| 阶段指标 Markdown | `03_DataRoom/generated/stage_metrics.md` | 写作时直接引用的指标表 |
| 当前数据索引 | `03_DataRoom/当前数据索引.md` | 说明数据口径和使用限制 |

## 使用规则

- 写作时优先引用 Markdown 指标表。
- 需要复算时读取 CSV。
- 不从外部记忆补数字。
- 发现缺口时用 `[DATA GAP]` 标注。
